SVG 寸法アノテーションの使い方

SVG の寸法線を素早く整えるための補助ツールです。背景色は SVG 内の色から自動抽出し、工業図面風の矢印と数値を重ねてプレビューできます。

基本操作

  1. 「SVG ファイルを読み込む」から SVG を選択するか、テキストエリアへ SVG コードを貼り付けます。
  2. 幅・高さ・単位の入力値を調整します。単位を変更しても数値は保持されます。
  3. 必要に応じて、アスペクト比の固定や背景透過、寸法ラベル、寸法値の丸めを切り替えます。
  4. プレビューで工業図面風の寸法線と数値を確認し、SVG 側を調整してください。

ヒント

  • 背景色は SVG の塗り／線の色から自動的に無彩色を選択します。背景を透過に切り替えることも可能です。
  • 寸法線は工業図面の矢印形状で描画され、数値は矢印の真上に表示されます。
  • 縦方向の寸法値は読みやすい向きになるよう、文字を回転して配置しています。
